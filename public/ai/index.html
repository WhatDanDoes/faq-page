AI Translations of The Holy Bible

King James Version

This project leverages the software developed by Meta's No Language Left Behind initiative.

The software and models were used to translate the King James Version of the Holy Bible from 1611 into modern French, German, and Russian using the same publicly-available pre-trained models as the NLLB researchers.

Translations:

The execution times listed below were while using an Nvidia GeForce RTX GPU.

French - 10.5 hours

German - 8.5 hours
